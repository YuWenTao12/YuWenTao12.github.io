Node.js版本切换（n或nvm）

2021-12-04 2 min read

[TOC]

n

n是Node的一个模块。

  • 安装：

    sudo npm install -g n

  • 版本切换：

    输入n，就会出现安装的node版本列表，如果前面有个"o"标识，表示这是你在当前正在使用的node版本

    n

     5.10.1 
o      6.2.2 
     7.0.0

  • 安装某个版本

    n x.x.x.x

  • 安装最新版本

    n latest

  • 安装稳定版本

    n stable

  • 删除某个版本

    n rm x.x.x.x

  • 指定某个版本来执行文件

    n user x.x.x some.js

nvm

nvm与n的实现方式不同，nvm是通过shell脚本实现的。

  • 安装：

    curl https://raw.github.com/creationix/nvm/v0.4.0/install.sh | sh
或
wget -qO- https://raw.github.com/creationix/nvm/v0.4.0/install.sh | sh
    • 上述两个脚本会把nvm库clone到/.nvm，然后在/.bash_profile、/.zshrc或/.profile末尾添加环境变量，source生效。

  • 安装某个版本node

    nvm install x.x.x

  • 使用指定版本

    nvm use x.x.x

  • 查看当前已经安装的版本

    nvm ls
.nvm
-> x.x.x

  • 查看正在使用的版本

    nvm current
vx.x.x

  • 指定某个版本来执行文件

    nvm run x.x.x some.js

  • 卸载nvm

    rm -rf ~/.nvm

        • nvm
